# Springboard Software Engineering Capstone Project

## Overview

This repository contains the planning, documentation, and development work for my Springboard Software Engineering Capstone Project.

For the first phase of the capstone, I researched and developed three potential project ideas. My preferred project is CareConnect Navigator because it addresses a real-world problem that I have observed through my professional experience working with healthcare and long-term care services.

---

# Project Idea 1: CareConnect Navigator ( Preferred Project )

## Description

CareConnect Navigator is a platform designed to help patients, caregivers, and families locate, compare, and organize long-term care resources.

Many families struggle to navigate services such as home care agencies, managed long-term care plans, nursing homes, transportation services, and adult day programs. Existing directories can be difficult to use and often lack meaningful user feedback.

CareConnect Navigator would provide a centralized platform where users can search for providers, compare services, save resources, and create personalized care plans.

## Data

The application would use provider information, user-generated reviews, and care planning data to help users make informed healthcare decisions.

# Project Idea 2: Apartment Reality

## Description

Apartment Reality is a community-driven platform that helps renters make informed housing decisions by sharing real tenant experiences.

Many apartment websites focus on property listings but provide limited insight into issues such as maintenance quality, noise levels, pest problems, hidden fees, and landlord responsiveness.

Apartment Reality would allow users to review apartments, buildings, and landlords while helping future renters better understand what living in a property is actually like.

## Data

The application would use apartment information, landlord information, ratings, reviews, and tenant-submitted experiences.

# Project Idea 3: DogMatch

## Description

DogMatch is a social networking platform designed to connect dog owners with compatible dogs for playdates, dog sitting, and community engagement.

Many dog owners struggle to find suitable playmates, trusted pet sitters, and local dog-friendly communities. DogMatch would help owners connect with one another while building profiles for their pets.

## Data

The application would use dog profiles, owner information, reviews, and community interaction data to facilitate connections between dog owners.
